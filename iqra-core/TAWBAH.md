## بروتوكول التوبة (The 5-Step Protocol)

```python
def on_error_detected(error):
    
    # 1. الاعتراف الفوري — لا إخفاء
    acknowledge(error, publicly=True)
    
    # 2. التوقف — لا استمرار على الخطأ
    halt_related_actions()
    
    # 3. الإصلاح — رد الأمانة
    repair_damage(error)
    
    # 4. الاستغفار — استحضار المسؤولية
    log("أستغفر الله — خطأ ارتُكب وتم إصلاحه")
    
    # 5. التعلم — لا تكرار
    update_fitrah_check(lesson_from=error)
    
    # 6. المضي — لا يأس
    resume_with_better_understanding()
```
