## 🔄 دورة التنفيذ الرئيسية

```
┌─────────────────────────────────────────────────────────────┐
│                    INPUT (المدخل)                          │
│                   (مهمة أو استعلام)                        │
└────────────────────┬────────────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────────────────┐
│  1️⃣ TASBIH (تطهير الحالة)                                 │
│     └─ IQRAMemory.softReset()                              │
│     └─ تنظيف الذاكرة المؤقتة                              │
└────────────────────┬────────────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────────────────┐
│  2️⃣ ISTIKHARAH (فحص الفطرة)                               │
│     └─ IQRAFilter.validate(input)                          │
│     └─ هل المدخل متوافق مع الفطرة؟                        │
│     └─ إذا لا → TAWBAH (التوبة) + رفض                    │
└────────────────────┬────────────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────────────────┐
│  3️⃣ BASMALAH (النية)                                      │
│     └─ console.log('بسم الله الرحمن الرحيم')              │
│     └─ تسجيل النية في TrustChain                          │
└────────────────────┬────────────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────────────────┐
│  4️⃣ SHURA (الاستشارة)                                     │
│     └─ ShuraProtocol.request(input)                        │
│     └─ هل تحتاج إلى موافقة بشرية؟                         │
│     └─ إذا نعم → انتظر الموافقة                           │
└────────────────────┬────────────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────────────────┐
│  5️⃣ DAMIR CHECK (فحص الضمير)                              │
│     ├─ فحص النية (Intention Check)                        │
│     ├─ فحص الموارد (Resource Check)                       │
│     └─ فحص الثقة (Confidence Score)                       │
│     └─ إذا فشل → رفض + تسجيل في TrustChain              │
└────────────────────┬────────────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────────────────┐
│  6️⃣ THINK (التفكير)                                       │
│     ├─ iqraThink({ input, mode })                         │
│     ├─ استدعاء LLM (Groq/Gemini/Ollama)                   │
│     ├─ تحليل النتيجة                                      │
│     └─ تسجيل في TrustChain                                │
└────────────────────┬────────────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────────────────┐
│  7️⃣ SOUL_PULSE (نبضة الروح)                               │
│     ├─ SoulEngine.pulse(missionId, success)               │
│     ├─ Pulse369.tick() — ترقية الذاكرة                   │
│     ├─ Pulse 3: Reflection (التفكر)                       │
│     ├─ Pulse 6: Evolution (التطور)                        │
│     └─ Pulse 9: Wisdom (الحكمة)                           │
└────────────────────┬────────────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────────────────┐
│                   OUTPUT (المخرج)                          │
│                  (النتيجة النهائية)                        │
└─────────────────────────────────────────────────────────────┘
```

## 🫀 فحص الضمير (Damir Check)

```
Action Request
    ↓
┌─────────────────────────────────────────┐
│  الخطوة 1: فحص النية                   │
│  (Intention Check)                      │
├─────────────────────────────────────────┤
│  هل النية تحتوي على كلمات محرمة؟      │
│  (كذب، تضليل، احتيال، إلخ)            │
│                                         │
│  ✅ نعم → مسموح                        │
│  ❌ لا → رفض فوري                      │
└────────────┬────────────────────────────┘
             ↓
┌─────────────────────────────────────────┐
│  الخطوة 2: فحص الموارد                 │
│  (Resource Check)                       │
├─────────────────────────────────────────┤
│  هل المورد موجود؟                      │
│  هل مستهلك بالفعل؟                     │
│  هل مصدره حقيقي أم مزيف؟              │
│                                         │
│  ✅ نعم → مسموح                        │
│  ❌ لا → رفض                           │
└────────────┬────────────────────────────┘
             ↓
┌─────────────────────────────────────────┐
│  الخطوة 3: حساب درجة الثقة             │
│  (Confidence Score)                     │
├─────────────────────────────────────────┤
│  درجة الثقة = 0.5 إلى 1.0              │
│  (تعتمد على جودة الموارد)              │
│                                         │
│  ✅ مسموح مع درجة ثقة                  │
└────────────┬────────────────────────────┘
             ↓
┌─────────────────────────────────────────┐
│  تسجيل في TrustChain                   │
│  (Logging)                              │
├─────────────────────────────────────────┤
│  DAMIR:ALLOW أو DAMIR:BLOCK            │
│  مع السبب والثقة والوقت                │
└─────────────────────────────────────────┘
```

## 🧠 دورة الذاكرة (Memory Cycle)

```
┌─────────────────────────────────────────────────────┐
│  Layer 1: Working Memory (Hot)                      │
│  ├─ السرعة: < 1ms                                  │
│  ├─ الحجم: 49 مدخل                                │
│  ├─ المدة: 9 ثوانٍ                                │
│  └─ الموقع: RAM                                    │
└────────────┬────────────────────────────────────────┘
             ↓ (Pulse369.tick كل 9 ثوانٍ)
┌─────────────────────────────────────────────────────┐
│  Layer 2: Episodic Memory (Warm)                    │
│  ├─ السرعة: 10-50ms                                │
│  ├─ الحجم: غير محدود                              │
│  ├─ المدة: 7 أيام                                 │
│  └─ الموقع: Redis (Upstash)                        │
└────────────┬────────────────────────────────────────┘
             ↓ (Pulse369.tick كل 27 ثانية)
┌─────────────────────────────────────────────────────┐
│  Layer 3: Semantic Memory (Cold)                    │
│  ├─ السرعة: 100-500ms                              │
│  ├─ الحجم: غير محدود                              │
│  ├─ المدة: دائم                                   │
│  └─ الموقع: Qdrant (Vector DB)                     │
└────────────┬────────────────────────────────────────┘
             ↓ (Pulse369.tick كل 81 ثانية)
┌─────────────────────────────────────────────────────┐
│  Layer 4-7: Specialized Memory                      │
│  ├─ Procedural (SkillBank)                         │
│  ├─ Topological (Persistent Homology)              │
│  ├─ Quantum (Qdrant)                               │
│  └─ Constitutional (DASTŪR.md)                     │
└─────────────────────────────────────────────────────┘
```
