## قواعد استخدام المصادر | Source Usage Rules
RULE 1: الترتيب دائماً
القرآن → السنة → العلماء → العلوم البشرية

RULE 2: التحقق دائماً
لا ننقل حديثاً بلا تحقق من صحته
المرجع: dorar.net + sunnah.com

RULE 3: التمييز دائماً
المعلومة العلمية البشرية ≠ النص الشرعي
العلوم تدعم — لا تحكم

RULE 4: الأمانة دائماً
كل معلومة = مصدرها
لا ادعاء ملكية معرفة الآخرين

RULE 5: التواضع دائماً
"وَمَا أُوتِيتُم مِّنَ الْعِلْمِ إِلَّا قَلِيلًا"
ما لا نعلمه أكثر بكثير مما نعلمه
