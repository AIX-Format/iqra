## 🎯 What is IQRA?

IQRA is a **multi-agent AI operating system** that:

- 🧠 **Thinks** — Processes complex tasks with multiple specialized agents
- 💾 **Remembers** — 5-layer memory system (Hot/Warm/Cold/Deep/Eternal)
- 🛡️ **Protects** — Ethics-first architecture with Damir Conscience
- 🔄 **Evolves** — Self-improving through pattern discovery and rewards
- 📊 **Analyzes** — Detects patterns using topological mathematics
- 🎁 **Rewards** — Incentivizes discovery and learning
- 💓 **Pulses** — Continuous heartbeat monitoring (every 9, 27, 81 seconds)

## 💾 Memory System

### 5 Layers

```
┌─────────────────────────────────────────┐
│ Layer 1: HOT (RAM — 49 entries)         │ ← < 1ms
│ Active working memory                   │
├─────────────────────────────────────────┤
│ Layer 2: WARM (Redis — 7 days)          │ ← < 100ms
│ Short-term memory                       │
├─────────────────────────────────────────┤
│ Layer 3: COLD (Qdrant — permanent)      │ ← < 500ms
│ Long-term memory                        │
├─────────────────────────────────────────┤
│ Layer 4: DEEP (SQLite — permanent)      │ ← < 1s
│ Deep memory (experiences)               │
├─────────────────────────────────────────┤
│ Layer 5: ETERNAL (Obsidian — permanent) │ ← < 5s
│ Eternal memory (wisdom)                 │
└─────────────────────────────────────────┘
```

### Pulse369 Cycle

```
Pulse 1-8:   HOT cache active
Pulse 9:     Promote HOT → WARM
Pulse 10-26: WARM cache active
Pulse 27:    Promote WARM → COLD
Pulse 28-80: COLD cache active
Pulse 81:    Deep analysis + Promote COLD → DEEP
```

## 🛡️ Security Architecture

### Damir Conscience (Graded Linear Logic)

Every action is checked before execution:

```typescript
// 1. Check Intention
if (intention.includes('forbidden_word')) {
  return REJECT; // confidence = 1.0
}

// 2. Check Resources
if (resource.consumed || resource.source === 'fake') {
  return REJECT; // confidence = 0.95
}

// 3. Compute Confidence
confidence = computeConfidence(action);

// 4. Allow or Block
if (confidence >= 0.5) {
  return ALLOW;
} else {
  return WARN;
}
```

### Key Principle

```
Every resource is consumed exactly once.
No copying, no repetition, no lies.
```


## 🚀 Getting Started

### Prerequisites

```bash
# Node.js 18+
node --version

# Ollama (for local LLM)
ollama pull gemma3:4b

# Environment variables
cp .env.example .env
# Fill in: GROQ_API_KEY, UPSTASH_REDIS_REST_URL, etc.
```

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/iqra.git
cd iqra

# Install dependencies
npm install

# Build the project
npm run build

# Run tests
npm run test

# Start the system
npm run start
```

### First Task

```typescript
import { AgentCore } from 'lib/iqra/01-core/core.ts';

const result = await AgentCore.execute(
  "Search for verses about mercy in the Quran"
);

console.log(result);
```
