# Knowledge Simulation — IQRA

This directory contains deterministic scenario material for running the Topological Curiosity loop without external API dependencies.

## Purpose

- Enable offline testing of resonance rewards.
- Use fixed modern-data scenarios with Quranic pattern anchors.
- Validate the system's behavior before connecting to live sources.

## Files

- `aya_resonance_scenario.md`: a sample of modern data mapped to Quranic patterns.
- `world_simulation.yaml`: mission inputs for a simulated resonance mission.
